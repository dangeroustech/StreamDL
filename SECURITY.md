# Security Policy

## Supported Versions

| Version     | Supported          |
| ----------- | ------------------ |
| >= 3.3.0    | :white_check_mark: |
| 3.0.0-3.2.x | :warning:          |
| < 3.0.0     | :x:                |

## Reporting a Vulnerability

Please file an issue using the SECURITY template, giving as much detail as possible.
